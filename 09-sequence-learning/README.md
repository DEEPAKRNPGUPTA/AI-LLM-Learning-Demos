# 09 — Sequence Learning

This demo is part of the **AI / LLM Learning Demos** series.

Sequence Learning is the bridge between **Embeddings** and **Attention**.  
After text becomes tokens and embeddings, the next question is:

> How does a model understand the order of words?

Because language is not just a collection of words.  
Order, memory, and context change meaning.
